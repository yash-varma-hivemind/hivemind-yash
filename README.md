# Hivemind Employee Onboarding

Nx monorepo containing a NestJS GraphQL API and Next.js web application.

## Applications

- `apps/api` — NestJS API with code-first GraphQL
- `apps/web` — Next.js, Tailwind CSS and shadcn/ui
- PostgreSQL — runs through Docker Compose
